# Makefile handles the compilation and execution!
**LIS library** (Library of Iterative Solvers for linear systems): 
  - Included in `../lis-2.1.10/` directory 
  - **Built automatically** by the Makefile when you run `make all`
  - No manual installation required - the Makefile handles configuration and compilation
  - Used for the tasks 7, 8 and 9
## Compilation
To compile the file just execute 
```bash
make all
``` 
and to run it
```bash
make run
```
To clean before compiling again
```bash
make clean
```
## Output
The complete output will be printed in terminal, and saved into `reports/challenge2_report.txt`
